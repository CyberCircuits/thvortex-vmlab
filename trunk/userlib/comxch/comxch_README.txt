COM Port Exchanger Components v1.0
----------------------------------------------

The comxch and comxchx components are both available in a normal and trace version.
The trace version of the components logs all COM port activity to VMLAB's Messages
window, while the normal version does not.

To install the normal version of the components:
   Copy "comxch.dll" and "comxchx.dll" to "C:\VMLAB\userlib"

To install the trace version of the components:
   Copy "trace\comxch.dll" and "trace\comxchx.dll" to "C:\VMLAB\userlib"
